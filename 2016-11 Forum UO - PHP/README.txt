**SOURCES NON DISPONIBLE**

Projet web (forum) réalisé en PHP avec le framework MVC pour un projet personel.

Nous étions une équipe de 2 programmeurs-analystes et utilisions la méthode de 
travail SCRUM.

Le projet couvre toute la réalisation d'un forum, allant des livrables jusqu'à 
la conception d'une première version fonctionnelle.

Mes tâches réalisées dans le projet:

- L'analyse
	- Diagramme d'architecture détaillié (100%)
	- Diagramme de contexte (100%)
	- Maquettes (30%)
	- Modèle physique (50%)

- Programmation (90% du back-end)
	- Database manager (back-end)
	- Tous les modèles
	- Manager de connexion
	- Manager de gestion d'utilisateur
	- Manager de la vue d'un profil
	- Manager du forum
	- Manager d'un post
	- Manager d'un commentaire

- Documentation
	- Rédaction du backlog du produit (50%)
	- Rédaction du backlog du sprint (50%)

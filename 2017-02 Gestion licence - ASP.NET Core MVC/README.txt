**SOURCES ET DOCUMENTATIONS DU PROJET NON DISPONIBLE POUR CAUSE DE CONFIDENTIALITÉ**

Projet web réalisé en ASP.NET, HTML5, CSS3 et JQuery pour MicroAge qui avait 
pour but principal de permettre la gestion des licences des différents logiciels 
de la compagnie.

J'étais seul comme programmeur-analyste et j'utilisais la méthode de travail 
SCRUM.

Le projet couvre toute la réalisation d'une application, allant de la 
réalisation d'un devis de conception jusqu'à la réalisation d'une première 
version.

Mes tâches réalisées dans le projet:

- Devis de conception (analyse)
	- Diagramme d'architecture détaillié
	- Diagramme de context
	- Maquettes
	- Modèle physique

- Base de données
	- Fichier SQL générant la base de données dans SQL Server
	- Procédure stocker en SQL pour gérer la sauvegarde des anciennes
	  licences

- Programmation
	- Database manager (back-end)
	- Gestion des sessions et variable de sessions
	- Page de connexion
	- Page de gestion d'utilisateur
	- Page de gestion de client
	- Page de gestion de module
	- Page de gestion de projet
	- Page de gestion de licence
	- Page de restauration de licence

- Documentation
	- Rédaction d'un devis de conception (proposition de système au client)
	- Rédaction du sprint rétrospective 1
	- Documentation des frameworks créés
		- Popup
		- DatabaseManager
